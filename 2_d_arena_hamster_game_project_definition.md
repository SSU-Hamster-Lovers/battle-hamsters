# 🐹 2D 아레나 햄스터 게임 - 프로젝트 정의 문서

---

# 1. 프로젝트 개요

## 프로젝트 방향
- 장르: 2D 사이드뷰 아레나 액션
- 플랫폼: 데스크톱 웹
- 멀티플레이: 실시간 PvP

## 핵심 컨셉
> 귀여운 햄스터들이 난전 속에서 서로를 밀어내고 탈락시키는 빠른 템포 PvP 게임

## 핵심 키워드
- 햄스터
- 귀여움
- 의외의 잔혹함
- 난전

---

# 2. 프로젝트 고정 원칙 (비가역적 결정)

1. Next.js 반드시 포함
2. 데스크톱 웹 게임
3. 키보드 + 마우스 조작
   - 키보드: 이동
   - 마우스: 방향 및 공격
4. 복잡한 스킬 시스템 금지
5. 전투는 무기 중심
6. 맵은 확장 가능 구조로 설계
7. 비개발자도 콘텐츠 제작 참여 가능해야 함
8. 성능 최적화를 위해 Rust 적극 검토

# 6. 맵 데이터 구조 (초안)

맵은 데이터 기반으로 구성한다.

- collision
- spawn_points
- item_spawn_points
- hazards
- decorations
- interactive_nodes (확장용)
