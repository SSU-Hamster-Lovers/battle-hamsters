# 아이템 설계

## 목표

아이템은 무기와 별개의 월드 스폰 축으로서, 전투 템포와 기동성, 생존성을 순간적으로 흔드는 요소다.
아이템은 플레이어를 더 많이 움직이게 만들고, 특정 지역을 둘러싼 경쟁을 만든다.

## 월드 스폰 구조

월드 스폰은 아래 두 종류로 나뉜다.

- `weaponSpawns`: 무기 공급 지점
- `itemSpawns`: 아이템 공급 지점

무기와 아이템은 같은 맵에 공존할 수 있지만 역할이 다르다.

- 무기 스폰: 전투 수단 자체를 공급
- 아이템 스폰: 이동/생존/상태를 보정

## 스폰 동작

- `fade_in` 계열 아이템은 필드에 **떠 있는 상태**로 유지될 수 있다.
- `airdrop` 계열 아이템은 공중에서 생성되어 **중력 영향을 받아 낙하**할 수 있다.
- 어떤 동작을 쓸지는 맵 데이터에서 정한다.

## 획득 방식

- 무기는 명시적 pickup(`E`) 기준을 유지한다.
- 아이템은 지나가며 즉시 획득하는 **자동 pickup**을 기본 기준으로 둔다.
- 이유는 아이템이 무기보다 더 빠르게 전투 리듬과 이동 선택을 흔드는 축이기 때문이다.

## MVP 아이템 종류

### 1. Speed Rank Up
- 효과: `moveSpeedRank` 증가
- 목적: 추격, 도주, 난전 템포 증가
- 규칙: 랭크는 `-7 ~ +7` 범위 안에서만 변한다.

### 2. Extra Life
- 효과: 생명 +1
- 목적: 장기 생존 보상
- 규칙: 최대 보유 생명 제한은 추후 밸런싱에서 결정

### 3. Health Recover
- 효과: HP 회복
- 목적: 전투 재진입 지원
- 규칙: 최대 HP를 초과해 회복할 수 없다.

### 4. Jump Boost
- 효과: 최대 점프 횟수 증가
- 목적: 기동 루트 확장, 고지대 접근, 전투 메타 변화
- 규칙:
  - 기본 점프 수는 `1`
  - 최대 점프 수는 `3`
  - 최소 점프 수는 `1`
  - 언더플로우/오버플로우를 허용하지 않는다.

## 점프 증가 규칙

점프 증가 아이템은 단순한 이동 보너스가 아니라 무기 메타를 흔드는 요소다.

- 근접/산탄 무기: 점프 증가 시 추격력 상승
- 빔/히트스캔 무기: 고지대 장악 및 추적 안정성 증가
- 맵 설계: 점프 아이템이 있으면 우회 루트와 빠른 재진입 루트를 고려해야 함

## 설계 원칙

1. 아이템은 무기보다 자주 바뀌는 전투 상태를 만든다.
2. 아이템은 지나치게 복잡한 스택 시스템보다 즉시 이해 가능한 효과를 우선한다.
3. 이동 관련 아이템은 게임의 빠른 템포를 강화해야 한다.
4. 아이템도 서버 authoritative 상태로만 반영한다.

## 향후 확장 후보

- 일시적 넉백 저항
- 일시적 무기 자원 보존 효과
- 스폰 감지 아이템
- 투척 강화 버프

단, 초기 단계에서는 이동/생존 중심 아이템만 다룬다.
