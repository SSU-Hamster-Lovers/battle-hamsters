# 코어 게임 규칙

## 인원

| 모드 | 인원 | 비고 |
|------|------|------|
| 개인전 | 2/4/8인 | 팀 없이 개별 경쟁 |
| 단체전 | 팀 A vs B vs C | 최대 3팀 |

기본값: 4인 개인전

## 승리 조건

### 포인트제 (기본)
- 기본 시간: 5분
- 상대를 처치할 때마다 포인트 획득
- 시간이 끝날 때 가장 높은 포인트가 승리
- 커스텀 매치에서 시간 조절 가능

### KO
- 상대방의 모든 생명을 소진시키면 승리

### 시간초과
- 시간이 끝날 때 HP가 가장 높은 순서로 승리
- HP 동일 시 무승부

## 전투 방식

### HP와 생명
- **HP**: 감소할 수 있으며, 0이 되면 사망
- **생명**: 기본 3개, HP가 0이 되면 1개 소모 후 부활
- **생명 0**: 패배 확정 (리프트/라스트맨 스탠딩 등)

### 데미지 특성
- **넉백**: 피격 시 밀려남
- **낙사**: 특정 구역에서 사망
- **즉사**: 일정 이상 데미지 시 즉시 사망

## 이동

- **스태미너**: 없음 (무제한 이동)
- **기본 속도**: 모든 캐릭터 동일
- **버프/디버프**: 랭크 시스템으로 속도 조절

## 랭크 시스템

포켓몬식 -7 ~ +7 등급.

| 랭크 | 이속 보정 |
|------|-----------|
| +7 | +70% |
| +6 | +60% |
| +5 | +50% |
| +4 | +40% |
| +3 | +30% |
| +2 | +20% |
| +1 | +10% |
| 0 | ±0% |
| -1 | -10% |
| -2 | -20% |
| ... | ... |
| -7 | -70% |

## 무기

### 무기 없는 경우
- 맨손 (격투)

### 스폰 방식
- **고정 스폰**: 특정 위치에 항상 존재 (나무 아래 코코넛 등)
- **랜덤 스폰**: 시간에 따라 무작위 위치에 생성
- **시작 무기**: 특정 맵에서 처음부터 특정 무기 지급

### 무기 관리
- 무기 버리기 기능 지원
- 무기 교체 가능

## 게임 흐름

```
로비 → 매칭 → 전투 → 결과 → 재시작
```
