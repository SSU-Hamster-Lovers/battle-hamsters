# 코어 게임 규칙

## 전투 철학

Battle Hamsters는 **빠르게 이동하고, 무기를 순환하며, 정신없이 난전이 벌어지는 아레나 PvP**를 지향한다.
정확한 수치는 추후 조정할 수 있지만, 아래 감각은 핵심 원칙으로 유지한다.

- 전투 템포는 빠르다.
- 가만히 각을 재기보다 계속 이동하는 쪽이 유리하다.
- 무기 확보와 위치 선점이 전투 흐름을 만든다.
- 낙사, 넉백, 고저차, 재진입 루트가 중요하다.
- 월드 스폰(무기/아이템) 순환이 전투 리듬을 만든다.

## 인원

| 모드 | 인원 | 비고 |
|------|------|------|
| 개인전 | 2/4/8인 | 팀 없이 개별 경쟁 |
| 단체전 | 팀 A vs B vs C | 최대 3팀 |

기본값: 4인 개인전

## 승리 조건

### 포인트제 (기본)
- 기본 시간: 5분
- 상대를 처치할 때마다 포인트 획득
- 시간이 끝날 때 가장 높은 포인트가 승리
- 커스텀 매치에서 시간 조절 가능

### KO
- 상대방의 모든 생명을 소진시키면 승리

### 시간초과
- 시간이 끝날 때 HP가 가장 높은 순서로 승리
- HP 동일 시 무승부

## 전투 방식

### HP와 생명
- **HP**: 감소할 수 있으며, 0이 되면 사망
- **생명**: 기본 규칙은 3개지만, 현재 플랫폼 이동 테스트 단계에서는 99개로 두고 반복 검증한다.
- **사망 후 리스폰**: 현재 테스트 규칙은 사망 후 3초 뒤 상공에서 다시 떨어지며 리스폰한다.
- **생명 0**: 최종 규칙에서는 패배 확정 (리프트/라스트맨 스탠딩 등)

### 데미지 특성
- **넉백**: 피격 시 밀려남
- **낙사**: 특정 구역에서 사망
- **즉사**: 일정 이상 데미지 시 즉시 사망

## 이동

- **스태미너**: 없음 (무제한 이동)
- **기본 속도**: 모든 캐릭터 동일
- **기본 이동 감각**: 빠른 템포를 지향
- **버프/디버프**: 랭크 시스템으로 속도 조절

### 플랫폼 이동 규칙
- `A / D` 또는 좌우 방향키: 좌우 이동
- `W` 또는 `Space`: 점프
- `S / 아래`: 공중에서는 급강하, 원웨이 플랫폼 위에서는 플랫폼 내려오기 입력으로 사용 가능
- 이동과 점프 판정은 서버 authoritative로 처리한다.

### 점프 규칙
- 기본 상태는 **1단 점프**
- 아이템이나 상태 효과에 따라 **2단 점프**, **3단 점프** 가능
- 점프 횟수는 **최대 3**, **최소 1**로 고정한다.
- 점프 확장은 캐릭터 고유 능력보다 **맵 순환과 기동성 강화 수단**으로 본다.

### 중력 / 낙사 규칙
- 플레이어는 중력 영향을 받는다.
- 바닥/플랫폼 착지 시 수직 속도는 초기화된다.
- 맵 아래 kill zone에 닿으면 낙사 처리한다.
- 테스트 단계에서는 낙사 후 3초 뒤 상공 spawn 위치에서 리스폰한다.

## 월드 스폰

월드 스폰은 크게 두 종류로 나눈다.

- **무기 스폰**: 전투용 무기 공급
- **아이템 스폰**: 기동성/생존성/전투 리듬을 흔드는 보조 자원 공급

### 아이템 스폰 예시
- 스피드 랭크 증가
- 생명 추가
- 체력 회복
- 점프 횟수 증가

## 무기

### 무기 없는 경우
- 기본 무기는 **맨손 (격투)**
- 기본 무기만 무제한 사용 가능

### 스폰 방식
- **고정 스폰**: 특정 위치에 항상 존재
- **랜덤 스폰**: 시간에 따라 무작위 위치에 생성
- **시작 무기**: 특정 맵에서 처음부터 특정 무기 지급 가능

### 무기 경제 규칙
- 기본 무기를 제외한 모든 무기는 **유한 자원 무기**다.
- 예비 탄약은 없다.
- 무기의 탄창/용량이 모두 소진되면 해당 무기는 종료된다.
- 무기를 다시 풀 상태로 쓰려면 **새 무기를 주워야 한다.**

### 무기 관리
- 무기 버리기 기능 지원
- 무기 교체 가능
- 바닥에 떨어진 무기와 스폰된 무기는 일정 시간 동안 획득되지 않으면 사라진다.
- 초기 기준 despawn 시간은 **약 10초**를 목표로 한다.
