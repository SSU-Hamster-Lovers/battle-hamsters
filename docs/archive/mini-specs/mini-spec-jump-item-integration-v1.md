# 미니 스펙 — 점프 아이템 연동 1차

## 작업명

점프 아이템 연동 1차

## 목표

현재 테스트 편의값으로 열어 둔 기본 3단 점프를 제거하고,
아이템 획득을 통해 점프 횟수가 변하는 실제 규칙으로 전환한다.

## 이번 범위

- 기본 `maxJumpCount = 1`로 복귀
- `jump_boost` 획득 시 점프 횟수 증가
- `1 ~ 3` clamp 적용
- 사망/리스폰 시 점프 횟수 유지 여부 규칙 확정
- HUD 또는 디버그 표시로 현재 점프 수 확인 가능하게 정리

## 비목표

- 캐릭터별 고유 점프 패시브
- wall jump / dash jump / glide
- duration 기반 점프 버프 만료 시스템

## 핵심 결정

### 1. 기본 점프 수는 다시 1로 돌린다

- 현재 3단 점프는 테스트용 편의값이다.
- 실제 규칙 기준선은 다시 `1`이어야 한다.

### 2. 점프 증가는 아이템 효과로만 올린다

- 서버가 pickup 시 `maxJumpCount`를 증가시킨다.
- 결과는 반드시 `1..3` 범위로 clamp한다.

### 3. 리스폰 시 초기화 여부를 명확히 한다

- 사망 시 점프 확장은 초기화
- 더 넓게는 **죽으면 모든 일반 전투 상태가 초기화**를 기본 규칙으로 둔다.
  - 속도 랭크업
  - 점프 횟수 증가
  - 무기
  - 이후 추가될 일반 버프/상태
- 단, “죽어야 발동하는 특수 규칙” 같은 예외성 메커니즘은 후속 설계에서 별도 정의한다.

## 완료 조건

- 기본 상태에서 1단 점프만 가능
- 아이템 획득 후 2단/3단 점프로 올라감
- 사망/리스폰 규칙이 문서와 코드에서 일치
- 죽으면 일반 전투 상태 초기화 원칙이 다른 스펙과 충돌하지 않는다

## 검증 방법

- 수동 테스트로 기본 1단 점프 확인
- 아이템 획득 후 점프 증가 확인
- clamp 3 초과 방지 확인
- 리스폰 후 상태 확인
