# Battle Hamsters

> 귀여운 햄스터들의 2D 아레나 PvP 게임

## 프로젝트 구성

| 앱 | 설명 | Tech Stack |
|----|------|------------|
| Portal | 포털, 로비, 매칭 | Next.js 14 |
| Game | 게임 클라이언트 | Phaser 3 |
| Server | 게임 서버 | Rust |

## 빠른 시작

### 전제 조건

- Node.js 20+
- Rust latest
- pnpm 8+

### 개발 환경 설정

```bash
# 클론
git clone https://github.com/SSU-Hamster-Lovers/battle-hamsters.git
cd battle-hamsters

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```
